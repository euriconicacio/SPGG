Download and Installation


SPGG download can be performed through the following link: http://www.cienciasgeodesicas.ufpr.br/spgg/. It is provided in a compressed format (.tar.gz).


To install it, the user shall follow these steps through the command line:


    1. After the download, uncompress the file spgg2.tar.gz in the folder you want to keep it. This task can be performed through command line, after accessing the directory where the downloaded file is, by the following code:


tar -xvzf spgg2.tar.gz



WARNING! IMPORTANT! - 777 PERMISSION IN THE TARGET DIRECTORY


Not only in order to uncompress the downloaded file, but also to enable SPGG correct operation, one must have write permission in the folder that contains SPGG. Since version 2.0, its operation demands reading, writing and executing permission in the folder that contains SPGG.



    1. Done! Your SPGG is ready for use!
