Getting Started


Starting SPGG


After a correct installation, the user must perform the following actions:


    1. In the folder where the uncompressing occurred, enter the folder dist created during installation. Such task can be performed through command line using the following code:


cd dist


    1. Run SPGG. Such task can be performed through command line using the following code:


./spgg2


    1. Done! In a few seconds, after performing some essential checks, SPGG will be started!



WARNING! IMPORTANT! - INTERNET CONNECTION


As stated in "System Requirements", it is essential that the computer in which one intends to use SPGG has full internet connection. Since this is one of the initial checks carried out by the application, in case there is no internet connection, it will be displayed an alert as the following one, and the application WILL NOT BE STARTED. As soon as the internet connection is reestablished, the user must execute SPGG start up procedure once again.



Figure 1 - Alert with error message due to lack of internet connection


In the other hand, in case there is plenty of internet connection, the following alert will be displayed:


Figure 2 - Alert with message confirming internet connection.


After started, the user will have access to all the features available at the application.
