System Requirements


SPGG was developed in Python programming language, with the main objective of being used in free operational systems. Thus, the essential requirements for its operation are:


    • Linux OS (preferably Debian 8 [Jessie] or Ubuntu 16.04 LTS); *
    • Chromium web browser (Google Chrome equivalent) installed; **
    • Access to the internet (preferably, cable internet access with connection speed of 100 Mbps or higher);
    • Spreadsheet editor (Microsoft Excel or OpenOffice Calc) - for input file formatting and output file visualization.


In addition, it is necessary for the user to have basic knowledge of Linux command line basic operation.


As SPGG is a Python3 programming language application, it works based on the following libraries, which are required for proper working: os, sys, time, glob, socket, datetime, webbrowser, collections, urllib.request, json, xlrd, xlwt, xlutils, pyvirtualdisplay, selenium and PyQt5.


* If the user do not have the above mentioned operating system, it is recommended the installation of a virtual machine to run the application. It is suggested the virtual machine manager Oracle VM VirtualBox and a OSGeo-Live virtual machine (Ubuntu 16.04) or pure Ubuntu 16.04.


A installation guide for installing the virtual machine manager and the virtual machine itself can be found at the following link: OSGeo-Live Quickstart for Running in a Virtual Machine.



** There may be issues due to Chromium installation and its conflicts with Google Chrome. Please, refer to "Troubleshooting" at the main menu.
