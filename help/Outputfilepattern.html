Output file pattern


The output file generated by SPGG is both simple and intuitive. It is generated a spreadsheet (".xls" file) with the same name of the input file, followed byt the suffix "_output", in the same folder where the input file is located. For example, if your input file is named points.xls and it is located in the folder /home/user - its full address is /home/user/pontos.xls. Thereby, the output file will be named points_output.xls and its full address will be /home/user/pontos_output.xls.


With regard to its content, the output file consists of two columns, as follows:


    • Column #1 - Id

Id number exactly as informed in the input file.


    • Column #2 - Result

The numercial result of the chosen functional, according to the provided data; e.g.: 3.346495696369
